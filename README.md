LibGodot Elixir
------------------------

Builds a connector for elixir with libgodot. For usage, see [libgodotconnector-starter](https://github.com/appsinacup/libgodotconnector-starter).

## How to build locally

Go to `elixir_sample` and follow instructions from there.
